# DevCore Core

[English] | [[日本語](README.ja.md)]

The "umbrella" artifact for using all DevCore modules together.

## Install (Gradle Kotlin DSL)

```kotlin
dependencies {
  implementation("com.peco2282.devcore:core:<version>")
}
```

## Included

- `adventure`
- `command`
- `config`
- `scheduler`
- `cooldown`
