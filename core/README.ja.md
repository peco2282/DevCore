# DevCore Core

[[English](README.md)] | [日本語]

DevCore の全モジュールをまとめて利用するための “umbrella” artifact です。

## Install (Gradle Kotlin DSL)

```kotlin
dependencies {
  implementation("com.peco2282.devcore:core:<version>")
}
```

## 含まれるモジュール

- `adventure`
- `command`
- `config`
- `scheduler`
- `event`
- `cooldown`
- `scoreboard-lite`
